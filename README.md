# week2-cycling

## 簡介

2021 F2E 挑戰賽第二週 - 自行車地圖資訊整合網站~~
主要使用 Vue3 Cli + Leaflet 地圖套件，搭配 SCSS 及 Bootstrap。
使用地圖套件解鎖，功能繼續學習中，第一版完成！

## 網站功能

- 全台 Ubike 站點查詢
- 即時 Ubike 站點租車/ 還車數量查詢
- 全台縣市自行車路線查詢
